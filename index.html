Q6: Top 10 Countries by Number of Fans

What are the top 10 countries considering the number of fans?

SELECT f.CountryCode, p.CountryName, f.NumberOfFans FROM FansPerCountry f LEFT JOIN PopStats p ON f.CountryCode = p.CountryCode WHERE f.Date = (SELECT MAX(Date) FROM FansPerCountry) ORDER BY f.NumberOfFans DESC LIMIT 10;
Result:
Returns top 10 countries with CountryCode, CountryName, and NumberOfFans

Q7: Top 10 Countries by Penetration Ratio

What are the top 10 countries by penetration ratio (% of population that are fans)?

SELECT p.CountryName, 1.0 * f.NumberOfFans / p.Population AS PenetrationRatio, f.NumberOfFans, p.Population FROM FansPerCountry f JOIN PopStats p ON f.CountryCode = p.CountryCode WHERE f.Date = (SELECT MAX(Date) FROM FansPerCountry) ORDER BY PenetrationRatio DESC LIMIT 10;

Top 10 countries by fan penetration percentage

Q9: Analysis by Age Group

What is the split of page fans across age groups (in %)?

Potential buying power based on US English-speaking fans (0.01% of income for subscriptions)

Section 5: Fan Engagement

Q14: Engagement per Day of Week

Split of EngagedFans across days of the week - Best day to publish?

WITH Base AS ( SELECT strftime('%w', CreatedTime) AS WeekdayNum, EngagedFans FROM PostInsights ) SELECT CASE WeekdayNum WHEN '0' THEN 'Sunday' WHEN '1' THEN 'Monday' WHEN '2' THEN 'Tuesday' WHEN '3' THEN 'Wednesday' WHEN '4' THEN 'Thursday' WHEN '5' THEN 'Friday' WHEN '6' THEN 'Saturday' END AS DayOfWeek, 100.0 * SUM(EngagedFans) / (SELECT SUM(EngagedFans) FROM Base) AS PercentageSplit FROM Base GROUP BY WeekdayNum ORDER BY WeekdayNum;
Result:
Engagement percentage by day of week

Q15: Engagement per Time of Day

Split of EngagedFans by time ranges - Best time to publish?

WITH B AS ( SELECT CASE WHEN time(CreatedTime) BETWEEN '05:00:00' AND '08:59:59' THEN '05:00 - 08:59' WHEN time(CreatedTime) BETWEEN '09:00:00' AND '11:59:59' THEN '09:00 - 11:59' WHEN time(CreatedTime) BETWEEN '12:00:00' AND '14:59:59' THEN '12:00 - 14:59' WHEN time(CreatedTime) BETWEEN '15:00:00' AND '18:59:59' THEN '15:00 - 18:59' WHEN time(CreatedTime) BETWEEN '19:00:00' AND '21:59:59' THEN '19:00 - 21:59' ELSE '22:00 or later' END AS TimeRange, EngagedFans FROM PostInsights ) SELECT TimeRange, 100.0 * SUM(EngagedFans) / (SELECT SUM(EngagedFans) FROM B) AS Percentage FROM B GROUP BY TimeRange ORDER BY TimeRange;
Result:
Engagement percentage by time of day

Section 6: Advanced Analysis

Q16: Month-to-Month Change in Engagement

Percentage changes in PostClicks, EngagedFans, and Reach month-over-month

WITH Monthly AS ( SELECT strftime('%Y-%m', CreatedTime) AS Month, SUM(PostClicks) AS PostClicks, SUM(EngagedFans) AS EngagedFans, SUM(Reach) AS Reach FROM PostInsights GROUP BY strftime('%Y-%m', CreatedTime) ) SELECT m1.Month AS FromMonth, m2.Month AS ToMonth, 100.0 * (m2.PostClicks - m1.PostClicks) / m1.PostClicks AS DeltaPostClicks, 100.0 * (m2.EngagedFans - m1.EngagedFans) / m1.EngagedFans AS DeltaEngagedFans, 100.0 * (m2.Reach - m1.Reach) / m1.Reach AS DeltaReach FROM Monthly m1 JOIN Monthly m2 ON m2.Month = ( SELECT MIN(Month) FROM Monthly WHERE Month > m1.Month ) ORDER BY FromMonth;
Result:
Month-over-month percentage changes in engagement metrics
